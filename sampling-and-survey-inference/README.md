
# Sampling and Survey Inference

This section contains technical notes, short papers, and applied examples in sampling and survey inference. The focus is on clear exposition, practical implementation, and reusable methodological reference material.

Topics include sampling designs, weighting, stratification, clustering, calibration, nonresponse, finite population inference, variance estimation, and related applied survey workflows. Notes may include theoretical background, simulation-based illustrations, and reproducible code in R and Python.
