
# Generalized Linear and Regression Models

This section contains technical notes, short papers, and applied examples in generalized linear models and regression modeling. The focus is on clear exposition, practical implementation, and reusable methodological reference material.

Topics include linear regression, logistic regression, Poisson and negative binomial models, ordinal and multinomial models, interaction effects, marginal effects, model interpretation, and related applied regression workflows. Notes may include theoretical background, simulation-based illustrations, and reproducible code in R and Python.
