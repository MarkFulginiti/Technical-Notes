
# Causal Inference

This section contains technical notes, short papers, and applied examples in causal inference. The focus is on clear exposition, practical implementation, and reusable methodological reference material.

Topics include identifiability, estimands, regression adjustment, propensity score methods, weighting, matching, heterogeneous treatment effects, and related applied causal workflows. Notes may include theoretical background, simulation-based illustrations, and reproducible code in R and Python.
