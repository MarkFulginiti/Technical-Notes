# Monte Carlo and Computational Methods

This section contains technical notes, short papers, and applied examples in Monte Carlo and computational methods. The focus is on clear exposition, practical implementation, and reusable methodological reference material.

Topics include Monte Carlo integration, importance sampling, variance reduction, accept-reject methods, Markov chain Monte Carlo, EM algorithms, numerical stability, and related computational workflows. Notes may include theoretical background, simulation-based illustrations, and reproducible code in R and Python.
